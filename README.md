 # 📰 Vintage Newspaper Project

A PHP-based news website with admin panel, articles system, and vintage UI.

## Features
- Article Management (CRUD)
- Admin Panel
- Login System
- Live Search
- Vintage UI Theme

## Tech Stack
- PHP
- MySQL
- HTML/CSS/JS

## Setup
1. Import database from /database/schema.sql
2. Run on XAMPP
3. Open: http://localhost/vintage-newspaper
